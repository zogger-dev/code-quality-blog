# Technology Stack

## Core Language & Runtime
- **Language:** Java 21 (LTS)
- **Rationale:** Using the latest LTS ensures examples leverage modern language features (Records, Pattern Matching, etc.) while remaining applicable to forward-looking enterprise environments.

## Content Format & Distribution
- **Primary Format:** Markdown (`.md`)
- **Distribution Strategy:** Portable Markdown files designed for easy integration into internal company repositories (e.g., GitHub/GitLab wikis, internal documentation sites, or engineering portals).

## Development & Verification Tools
- **Code Formatting:** Google Java Style (Baseline)
- **Editor Recommendation:** Any Markdown-compatible editor (IntelliJ IDEA, VS Code, or Obsidian are recommended for integrated Java/Markdown support).
